About

The data of this project was legally obtained from the Open Data City of Windsor page.

See below the definition of each variable:

  • XCoord: Latitude. Type: Float
  • YCoord: Longitude. Type: Float
  • FID Sample: Sample identification number. Type: INT
  • Species: Specie of the tree. Type: TXT
  • DBH: Diameter at Breast Height (in centimeters). Type: Float
