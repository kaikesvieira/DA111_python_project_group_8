About

This project uses data legally obtained from the Open Data City of Windsor page.

Below is the definition of each variable used in the dataset:

  • XCoord: Latitude. Type: Float
  • YCoord: Longitude. Type: Float
  • FID Sample: Sample identification number. Type: INT
  • Species: Species of the tree. Type: TXT
  • DBH: Diameter at Breast Height (in centimeters). Type: Float
Data source: Open Data City of Windsor.
